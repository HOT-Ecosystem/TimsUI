Simple Query FHIR Servers
If you get no response from the forms in the pages linked above, for reasons related to simple deployment of this code, you'll need to try one of two solutions below. Future deployments will fix this.

1. Copy the file local

  • Load this page: Simple Query FHIR Servers
  • From the File menu, select "Save Page As..."
  • Enter "query.html", select save
  • Again from the File menu, select "Open file..."
  • Select "query.html", select open
  • proceed

2. Setting Chrome so it allows calling an http page from one loaded as https:

from Stack Overflow

On OSX using Chrome

Tested on build (2/20/2020, 79.0.3945.130)
  • Click on the 'i' info,lock icon, or a red triangle, on the left side of address bar.
  • Click Site Settings
  • Scroll down to Insecure content
  • Change it from Blocked (Default) to Allow
  • Reload the page and try your action again.
  • Repeat this later, setting it back to Blocked, to turn it back on if you like.

On Windows using Chrome

Tested on version 108.0.5359.99
  • Click on the 'i' info,lock icon, or a red triangle, on the left side of address bar.
  • Click Site Settings
  • Scroll to Privacy and security
  • Scroll down to Insecure content
  • Change it from Blocked (Default) to Allow
  • Reload the page and try your action again.
  • Repeat this later, setting it back to Blocked, to turn it back on if you like.
